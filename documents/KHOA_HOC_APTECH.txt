Khóa học Aptech tại CUSC đào tạo lập trình viên quốc tế chuyên về lập trình web và phát triển phần mềm theo tiêu chuẩn quốc tế. Khóa học khai giảng ngày 28/05/2024, học vào các buổi tối thứ 3, 5, 7 từ 18h đến 21h. Thời gian đăng ký từ 11/01/2023 đến 28/05/2024.---
---
Chương trình Aptech tại CUSC gồm 4 học kỳ, đào tạo lập trình viên chuyên nghiệp với kiến thức và kỹ năng toàn diện về phát triển phần mềm, từ lập trình web, ứng dụng desktop, đến ứng dụng doanh nghiệp và công nghệ di động hiện đại, 4 học ký đó là:
Học kỳ 1: Phát triển ứng dụng web tương thích đa thiết bị.
Học kỳ 2: Lập trình ứng dụng Windows.
Học kỳ 3: Ứng dụng doanh nghiệp trên nền điện toán đám mây.
Học kỳ 4: Ứng dụng doanh nghiệp với công nghệ Java EE.

---

Học kỳ 1 có tên là Phát triển ứng dụng web tương thích đa thiết bị.
Học kỳ 1 sẽ giúp học viên xây dựng nền tảng lập trình bằng C, thiết kế giao diện website hiện đại, tạo trải nghiệm người dùng tốt qua UI/UX và lập trình web đa thiết bị. Học viên được học sử dụng SQL Server để quản lý dữ liệu, phát triển frontend bằng React và backend bằng PHP với Laravel. Cuối kỳ, học viên thực hiện đồ án xây dựng website hoàn chỉnh để rèn luyện kỹ năng tổng hợp và làm việc nhóm.

---

Học kỳ 2 có tên là Lập trình ứng dụng Windows
Học kỳ 2 sẽ tập trung phát triển kỹ năng lập trình hướng đối tượng với Java và C#. Học viên học cách xử lý dữ liệu bằng XML và JSON, xây dựng ứng dụng Java từ cơ bản đến nâng cao và phát triển ứng dụng Windows với C#. Cuối kỳ, đồ án Java giúp học viên củng cố kiến thức và rèn luyện tác phong làm việc nhóm chuyên nghiệp.

---

Học kỳ 3 có tên là Ứng dụng doanh nghiệp trên nền điện toán đám mây
Học kỳ 3 sẽ giúp học viên học phát triển ứng dụng NodeJS, Python, ASP.NET Core MVC, làm việc với dữ liệu lớn bằng MongoDB và triển khai trên Microsoft Azure. Cuối kỳ thực hiện đồ án sử dụng nền tảng .NET.

---

Học kỳ 4 có tên là Ứng dụng doanh nghiệp với Java EE
Học kỳ 4 sẽ tập trung phát triển ứng dụng web chuyên sâu với Jakarta EE và Spring Framework, lập trình mobile đa nền với Flutter/Dart, xây dựng dịch vụ web, kết thúc bằng dự án lập trình Java EE và kỳ thực tập tại doanh nghiệp hoặc CUSCSoftware.
